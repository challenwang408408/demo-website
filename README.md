# 中国历史五千年 - 赛博朋克版

一个展示中国历史朝代更替的赛博朋克风格网站。

## 功能特点

- 🎨 赛博朋克风格设计
- 📜 完整的历史朝代时间轴
- 🏛️ 重点朝代详细展示（秦汉隋唐宋明清）
- ✨ 炫酷的视觉效果和动画
- 📱 响应式设计，支持移动端

## 技术栈

- HTML5
- CSS3 (赛博朋克风格)
- JavaScript
- Flask (Python Web框架)

## 运行方式

### 本地开发

```bash
pip install -r requirements.txt
python app.py
```

访问 http://localhost:5000

### Docker部署

```bash
docker build -t chinese-history .
docker run -p 5000:5000 chinese-history
```

## 部署

使用 ai-builders-coach 工具进行部署上线。
